# Tobigticon
## team sol1: conditional video generation - landmark generation

- landmark detection
- landmark generation
- face warping
- face morphing

전체 프로젝트 코드는 [여기](https://github.com/Tobigs-team/EmoGET_tobigticon)에서 확인하실 수 있습니다.

## Requirements
We have tested on:
``` 
- CUDA 11.0
- python 3.8.5
- pytorch 1.7.1
- numpy 1.19.2
- opencv-python  4.5.1
- dlib 19.21.1
- scikit-learn 0.24.0
- Pillow 8.1.0
- Ninja 1.10.0
- glob2 0.7
```
